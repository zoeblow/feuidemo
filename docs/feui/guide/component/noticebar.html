API

参数 说明 类型 默认值 可选值
text 滚动显示的文字 String - -
type 通知栏右边图标样式 String - beclosed，link
leftImg 左侧图标图片链接,优先级大于leftIcon String - -
leftIcon 左侧字体图标类型，与leftImg任选其一即可 String - -
color 文本颜色 String #FF5B00 -
background 背景颜色 String #FFE1AF -
delay 动画延迟时间，单位秒 Number 1 -
speed 滚动速率，单位px Number 50 -
